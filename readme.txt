
<< 구성 >>

src/predictor.py  -> 핵심 모듈. 학습 및 테스트
src/preprocessor.py -> 데이터 전처리기
src/data_generator.py -> 데이터 생성기(processing module을 사용하여, 경매데이터를 이용해서 국토부 데이터를 순회)
이미 데이터는 전처리되어 batch_data_h5py에 저장되어 있으므로 참고용으로만 사용.
best_model/ -> 제일 좋은 모델 저장 기록.


<< 실행 방법 >>

1. pip_list 설치.
  $ pip3 install -r pip_list.txt

2.커맨드 라인에 다음과 같이 입력
~/price_rnn$ python3 src/predictor.py
 입력. 
 2015년도부터 2017년 2월 데이터를 이용하여 shuffle된 데이터로 학습
 epoch등 하이퍼파라미터는 파일 내에서 수정 / Test와 Train는 파일 내에서 flag로 수정.



<< 개발 환경 및 언어 >>
Ubuntu 16.04.3 LTS (GNU/Linux 4.10.0-32-generic x86_64)
python3 & tensorflow-gpu
